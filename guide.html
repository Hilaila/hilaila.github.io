Étape 1 — Ouvrir GeoTerraChain QFS
Accéder à la plateforme dans Pi Browser
COMPLÉTÉ
1
Ouvrir Pi Browser sur votre téléphone
Pi Browser est l'application officielle de Pi Network pour accéder aux apps. Téléchargez-le sur App Store ou Google Play si ce n'est pas encore fait.
2
Taper l'URL dans la barre d'adresse
Dans la barre d'adresse en haut de Pi Browser, tapez exactement :
https://hilaila.github.io
◈Le splash screen GeoTerraChain QFS apparaît pendant 2 secondes, puis l'app se charge automatiquement.
3
Vérifier que le Pi SDK est initialisé
Dans le header de l'app, vous devez voir :
● Pi SDK v2.0 ✓
✓Si vous voyez "● Hors Pi Browser", utilisez bien Pi Browser et non Chrome ou Firefox.

Étape 2 — Se connecter avec votre compte Pi
Authentification Pioneer KYC via Pi SDK 2.0
EN COURS
KYC obligatoire — Seuls les Pioneers dont le KYC est approuvé par Pi Network peuvent accéder à la tokenisation. La vérification est automatique lors de la connexion.
1
Cliquer sur "π Connexion" dans la navigation
La barre de navigation en haut de l'app contient 6 onglets. Cliquez sur le deuxième onglet.
π Connexion
2
Cliquer sur "Authentifier + Vérifier KYC"
Le bouton violet lance le processus d'authentification Pi SDK. Pi Network va vérifier automatiquement votre identité KYC.
◎Une fenêtre Pi Network s'ouvre pour demander votre autorisation. Cliquez sur "Allow" pour autoriser GeoTerraChain QFS à accéder à votre compte.
3
Vérifier le statut KYC dans le header
Après connexion réussie, vous devez voir dans le header :
KYC ✓ ● VotreUsername
✓Si vous voyez "KYC ✗", votre vérification KYC Pi Network n'est pas encore complétée. Rendez-vous dans l'app Pi Mining → KYC pour finaliser.

Étape 5 — Comprendre le calcul burn
Mécanisme déflationnaire GTCπ automatique
À FAIRE
Burn déflationnaire — À chaque tokenisation, 0,5% du prix est automatiquement brûlé (détruit définitivement). Cela rend GTCπ déflationnaire et augmente sa valeur au fil du temps.
Exemple avec un actif à 100 GTCπ :
CALCUL ON-CHAIN AUTOMATIQUE
Prix brut de l'actif100 GTCπ
Burn déflationnaire (0,5%)− 0.5 GTCπ
Net reçu par le propriétaire99.5 GTCπ
Calculateur interactif :
GTCπ → Burn: 0.5 · Net: 99.5 GTCπ

Questions fréquentes
Pourquoi le KYC est-il obligatoire ? +
Le KYC (Know Your Customer) garantit que chaque tokenisation est effectuée par un humain réel et vérifié. Cela protège l'intégrité du registre foncier on-chain et prévient la fraude. Pi Network vérifie votre identité via une pièce officielle et un selfie avant d'approuver votre compte Pioneer.
Qu'est-ce que le burn déflationnaire de 0,5% ? +
À chaque tokenisation, 0,5% du montant en GTCπ est détruit définitivement. Ce mécanisme réduit progressivement la supply totale de GTCπ (3 141 592), ce qui augmente la rareté et donc la valeur du token au fil du temps. C'est un mécanisme économique conçu pour bénéficier aux détenteurs de GTCπ sur le long terme.
Quels types d'actifs peut-on tokeniser ? +
GeoTerraChain QFS accepte 9 types d'actifs : Cultures vivrières, Cultures de rente (cacao, café, hévéa, palmier), Élevage et pastorale, Aquaculture et pisciculture, Agroforesterie et sylviculture, Maraîchage et horticulture, Immobilier urbain, Commercial, et Résidentiel. Tous les actifs situés dans la zone UEMOA sont éligibles.
Que signifie "on-chain" ? +
"On-chain" signifie que votre tokenisation est enregistrée de façon permanente et immuable sur le ledger (registre) Stellar — la blockchain sous-jacente de Pi Network. Chaque token possède un hash de transaction unique, vérifiable publiquement. Personne ne peut modifier ou effacer cet enregistrement.
Combien de Test-Pi dois-je avoir pour tokeniser ? +
Vous avez besoin d'au moins le montant du prix en GTCπ + 0,01 Pi de frais de transaction. Pour un premier test, utilisez 1 GTCπ — vous aurez besoin d'environ 1,01 Test-Pi dans votre wallet. Vous pouvez obtenir des Test-Pi gratuitement via developer.minepi.com/faucet dans Pi Browser.
Comment voir mes tokenisations passées ? +
Rendez-vous dans l'onglet "◇ Portefeuille" de l'app. Toutes vos tokenisations et transactions sont conservées et affichées avec les détails complets : hash de transaction, ledger Stellar, montant, burn, et timestamp.
